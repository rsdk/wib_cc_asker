# Asker
1. Im Backend eine Frage mit festen Antwortmöglichkeiten stellen.
2. Studenten klicken auf der öffentlichen Frage-Webseite auf eine Antwort.
3. Diagramm mit Verteilung der Antowrten im backend anschauen.
